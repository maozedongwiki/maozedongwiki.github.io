# 十六字令·三首
1934-1935年

山，  
快马加鞭未下鞍。  
惊回首，离天三尺三。

山，  
倒海翻江卷巨澜。  
奔腾急，万马战犹酣。

山，  
刺破青天锷未残。  
天欲堕，赖以拄其间。

</font>

</div>

# 原注
民谣：“上有骷髅山，下有八宝山，离天三尺三。人过要低头，马过要下鞍。”

# 题记
作于二万五千里长征途中湘贵一带群山间。

# 注释
锷（e4）：剑刃。  
天欲堕，赖以拄其间：《淮南子·天文训》，“昔者共工与颛顼争为帝，怒而触不周之山，天柱折，地维绝。”神话中有以山为撑天之柱的讲法.
