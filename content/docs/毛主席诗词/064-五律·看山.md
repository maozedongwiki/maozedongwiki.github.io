# 五律·看山
1955年

三上北高峰，杭州一望空。  
飞凤亭边树，桃花岭上风。  
热来寻扇子，冷去对美人。  
一片飘飘下，欢迎有晚莺。

</font>

</div>

# 题记
这首诗最早发表在《党的文献》一九九三年第六期。

# 注释
北高峰：在浙江省杭州市灵隐寺后，与南高峰相对峙，为西湖群山之一。在北高峰附近有飞凤亭、桃花岭、扇子岭、美人峰等名胜。
根据作者自注，诗中的“扇子”指扇子岭，“佳人”指美人峰。  
飘飖：同“飘摇”，飘荡、飞扬貌。这里指鹰翔。
