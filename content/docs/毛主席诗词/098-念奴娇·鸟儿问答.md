# 念奴娇·鸟儿问答
1965秋

鲲鹏展翅，九万里，翻动扶摇羊角。  
背负青天朝下看，都是人间城郭。  
炮火连天，弹痕遍地，吓倒蓬间雀。  
怎么得了，哎呀我要飞跃。

借问君去何方，雀儿答道：有仙山琼阁。  
不见前年秋月朗，订了三家条约。  
还有吃的，土豆烧熟了，再加牛肉。  
不须放屁！试看天地翻覆。

# 注释
鸟儿问答：见《庄子·逍遥游》和《七律·吊罗荣桓同志》的注释。  
三家条约：指美英苏三国签署的部分禁止核试验条约。  
土豆、牛肉：六四年四月访问匈牙利时，苏共领导赫鲁晓夫曾说，共产主义就是人人有土豆牛肉汤喝。
