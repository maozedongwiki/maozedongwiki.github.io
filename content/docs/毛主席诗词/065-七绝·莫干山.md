# 七绝·莫干山
1955年

翻身复进七人房，  
回首峰峦入莽苍。  
四十八盘才走过，  
风驰又已到钱塘。

</font>

</div>

# 题记
这首诗最早发表在《党的文献》一九九三年第六期。

# 注释
莫干山：在浙江省德清县西北。相传春秋时吴国在此铸“莫邪”、“干将”二剑，故名。为浙北避暑、休养胜地。  
七人房：指作者使用的卧车，可坐七人。  
四十八盘；泛写曲折盘旋的山间公路。  
钱塘：旧县名。这里指杭州市。
