# 如梦令·元旦
1930年1月

宁化、清流、归化，  
路隘林深苔滑。  
今日向何方，  
直指武夷山下。  
山下山下，  
风展红旗如画。

</font>

</div>

# 题记
毛泽东复出后，在二九年十二月二十八日至三十日召开了古田会议，整顿了红军，整肃了不同意见。在三零年初他行事甚为顺利。这首词和下一
首词作于这段心情愉快的时期。

# 注释
如梦令：词牌名，得名于唐庄宗诗句“如梦、如梦”。  
宁化、清流、归化：福建县名。  
武夷山：在福建北部。
