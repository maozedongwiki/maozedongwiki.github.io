# 五律·海鸥将军千古
1939年7月

外侮需人御，  
将军赋采薇。  
师称机械化，  
勇夺虎罴威。

浴血东瓜守，  
驱倭棠吉归。  
沙场竟殒命，  
壮志也无违。

# 悼戴安澜

外侮需人御，将军赴采薇。 师堪机械化，勇夺虎罴归。 浴血东瓜守，逐倭棠古回。 沙场竟殒命，壮志也无违。 据称是毛泽东与朱德悼戴将军的挽联。 
只见于将军决战岂止在战场一书中，真伪不知。
