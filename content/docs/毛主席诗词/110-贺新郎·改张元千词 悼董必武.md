# 贺新郎·改张元千词 悼董必武
1975年4月

梦绕神州路，怅秋风，连营画角，故宫离黍。  
底事昆仑倾砥柱，九地黄流乱注?聚万落千村狐兔。  
天意从来高难问，况人情易老悲难诉。  
更南浦，送君去。

凉生柳岸催残暑，耿斜河，疏星淡月，断云微度。  
万里江山知何处，回首对床夜语。雁不到，书成谁与?  
目尽青天怀今古，肯儿曹恩怨相尔汝!  
君且去，休回顾。
