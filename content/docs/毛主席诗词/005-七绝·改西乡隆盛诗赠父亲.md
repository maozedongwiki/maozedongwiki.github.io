# 七绝·改西乡隆盛诗赠父亲
1909年秋

孩儿立志出乡关，学不成名誓不还。  
埋骨何须桑梓地，人生无处不青山！

</font>

</div>

# 题记
改西乡隆盛诗，源于《毛泽东自传》。  
1910年，毛泽东的父亲毛顺生要他去做生意，毛泽东却立志走出韶山冲到新式学校湘乡县立东山高等小学继续求学。经过自己的力争和亲友、老师们的一致
劝说，父亲答应他的要求。在离家赴前夕，毛泽东提笔写了一首《赠父诗》，夹在父亲每天必看的账簿里。这首诗是少年毛泽东走出乡关、奔向外面世界的
宣言书，表明了他胸怀天下、志在四方的远大抱负。
