# 浣溪沙·和柳亚子先生
1950年10月

一九五零年国庆观剧，柳亚子先生即席赋《浣溪沙》，因步其韵奉和。  
长夜难明赤县天，  
百年魔怪舞翩跹，  
人民五亿不团圆。

一唱雄鸡天下白，  
万方乐奏有于阗，  
诗人兴会更无前。

</font>

# 注释
浣溪沙：词牌名，唐教坊（音乐学校）里曲子的名称。  
赤县：战国时称中国为赤县神州。  
翩跹（pian1　qian1）：跳舞轻盈状。  
一唱雄鸡天下白：李贺《致酒行》，“雄鸡一声天下白”。  
万方：古人称国族为方，《易经》有“鬼方”。  
于阗（tian2）：汉朝西域国名，在今新疆。  
兴会：兴致、兴趣。  

# 附 柳亚子原词

火树银花不夜天，  
弟兄姐妹舞翩跹，  
歌声唱彻月儿圆。

不是一人能领导，  
那容百族共骈阗，  
良宵盛会喜空前。
